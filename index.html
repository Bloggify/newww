What is Bloggify?

Bloggify is a NodeJS platform that provides a simple way to build a blog in minutes. Few of the advantages are listed below, the main ones being simplicity and speed. The development started in April, 2014 and the platform will be open sourced soon! Your support can help that.

Plugins

Bloggify is like a puzzle: it is built from multiple pieces. Each piece must do one thing and do it good. And these pieces are the plugins. Even most of the Bloggify core parts are plugins. That makes maintaining a lot easier. The concept is: Throw away what you don't use. You don't need a feature? Cool. Just uninstall the plugin, you won't use.

For example, the Bloggify dashboard is a plugin. If you don't need a dashboard in your browser, just use a Bloggify client application for your platform and uninstall dashboard plugin. If you are a geek, you can always do the things from command line and not use any user interface.

Features

Simplicity

Bloggify is simple to use. With a minimalist interface, you can do anything you expect from a blogging platform. If you still want more, you can install plugins, or if you're a developer, create plugins.

It's fast!

Bloggify's performance is awesome. Caching techniques are used (both on server and client) to speed up the things.

API

Bloggify comes a nice REST API for developers. Using this API developers can create third-pary applications to access Bloggify resources.

Git Integration

Bloggify's API has built-in Git integration. This is cool when talking about content revisions. All changes made by user are saved and commited into a git repository. This repository can be synced with a Git provider (e.g. GitHub), so other people will be able to contribute to content (e.g. fixing typos etc).

Themes

Themes are funny! As human, you can choose a theme you like. As developer, you can create a theme in few minutes. The Bloggify themes contain jade, html and css files. Also, they contain the package.json file that has NPM and theme information. And yes, themes support NodeJS modules!
