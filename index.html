What is Bloggify?

Bloggify is a NodeJS framework that provides a simple way to build a blog in minutes. Some of the advantages are listed below, the main ones being modularity, simplicity and speed.

The Bloggify core is not public yet, but some of the tools that power it (including themes, plugins and other tools) are open-source here.

We intend to release a stable release which will be free and open-source. You can help that!

Features

Simplicity

Bloggify is simple to use. With a minimalist interface, you can do anything you expect from a blogging platform. If you still want more, you can install and create plugins easily.

Speed

Bloggify's performance is awesome. Caching techniques are used (both on server and client) to speed up the things.

API

Bloggify has a nice REST API for developers. Using it, the developers can create third-pary applications to access the Bloggify resources.

Git Integration

Bloggify's API has built-in Git integration. This is cool when talking about content. All changes made by user are saved automatically into a git repository. This repository can be synced with a Git provider (e.g. GitHub), so other people will be able to contribute to content (e.g. fixing typos and so on).

Plugins

Bloggify is like a puzzle: it is built from multiple pieces. Each piece must do one thing and do it well. These pieces are called plugins. Bloggify itself is just a tiny server that puts them together. That makes maintaining the tools easier. The concept is: Throw away what you don't need. Do you not need a feature? Cool. Just uninstall it by removing the plugin.

For example, the Bloggify dashboard is a plugin. If you don't need a dashboard in your browser, just uninstall it. You can use a Bloggify client for your platform. If you are a geek, you can always do the things from command line and not use any user interface.

Themes

The themes are another part that makes Bloggify awesome. If you are non-technical, you can choose a theme you like. However, as developer, you can create a theme easily.
